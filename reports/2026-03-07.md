## Executive Summary

- Institution filters: Carnegie Mellon University; Massachusetts Institute of Technology; Stanford University; ETH Zurich; UC Berkeley; UC Berkeley; New York University; UC San Diego  
- Topic filters: mobile manipulation; motion planning; dextrous manipulation; whole-body control  

- Two institution-matched works appear today: one on mechanistic steering of vision-language-action (VLA) models for robot behavior (Stanford) and one on sim-to-real tactile dexterous manipulation (CMU/UC Berkeley).
- Whole-body control and locomotion/manipulation integration are advanced by GPU-native motion generation (cuRoboV2) and physics-aware humanoid VLA control (PhysiFlow).
- Dextrous and bimanual manipulation is pushed by large-scale synthetic datasets and policies for universal dexterous grasping (UltraDexGrasp) and sim-to-real tactile latent distillation (PTLD).
- Motion planning and mobile manipulation are addressed with dynamics-aware sampling-based planning (GAIDE, Many-RRT*), risk-aware mobile manipulation policies, and contact-tolerant navigation using VLMs.
- Several works target long-horizon, memory- and language-conditioned manipulation (SeedPolicy, VPWEM, HD-ExpIt, Critic-in-the-Loop, SkillsCrafter, RoboMME), highlighting temporal reasoning and hierarchical control as emerging themes.

| Observing and Controlling Features in Vision-Language-Action Models | Stanford University | Introduces feature-observability and feature-controllability for VLAs, learning linear feature detectors in representation space and applying optimal-control-style linear interventions to steer internal activations. Demonstrates that small targeted edits can reliably redirect robot behavior without fine-tuning, preserving closed-loop capabilities across different VLA architectures. | https://arxiv.org/abs/2603.05487 |
| PTLD: Sim-to-real Privileged Tactile Latent Distillation for Dexterous Manipulation | Carnegie Mellon University; UC Berkeley | Proposes a sim-to-real framework where dexterous manipulation skills are learned in simulation and then augmented with real-world tactile information via privileged sensing and latent distillation. A tactile state estimator is trained from real data to condition the policy on tactile input, yielding large performance gains in in-hand rotation and reorientation tasks over proprioception-only baselines. | https://arxiv.org/abs/2603.04531 |

| Risk-Aware Reinforcement Learning for Mobile Manipulation |  | Trains distributional RL teacher policies with distortion risk metrics to obtain families of risk-sensitive whole-body mobile manipulation behaviors in cluttered environments. Then distills these into depth-conditioned visuomotor policies, enabling runtime-adjustable risk levels and improved worst-case performance while navigating and manipulating. | https://arxiv.org/abs/2603.04579 |

| PhysiFlow: Physics-Aware Humanoid Whole-Body VLA via Multi-Brain Latent Flow Matching and Robust Tracking |  | Proposes a multi-brain VLA architecture that couples semantic motion intent with physics-aware whole-body control for humanoids. The framework uses latent flow matching and robust tracking to translate language-guided visual inputs into coordinated full-body behaviors for tasks like walking, sitting, and arm motions. | https://arxiv.org/abs/2603.05410 |
| UltraDexGrasp: Learning Universal Dexterous Grasping for Bimanual Robots with Synthetic Data |  | Introduces UltraDexGrasp-20M, a 20M-frame synthetic dataset of physically plausible bimanual dexterous grasps generated via optimization and planning, across multiple grasp strategies. Trains a point-cloud-based policy with unidirectional attention to output control commands, achieving robust zero-shot sim-to-real dexterous grasping on diverse real objects. | https://arxiv.org/abs/2603.05312 |
| Omni-Manip: Beyond-FOV Large-Workspace Humanoid Manipulation with Omnidirectional 3D Perception |  | Proposes a LiDAR-driven 360° visuomotor policy for humanoid manipulation in large workspaces, using a Time-Aware Attention Pooling module to encode panoramic point clouds with temporal context. A whole-body teleoperation system collects training data, and the learned policy outperforms egocentric depth-camera baselines in cluttered, wide-area manipulation scenarios. | https://arxiv.org/abs/2603.05355 |
| Risk-Aware Rulebooks for Multi-Objective Trajectory Evaluation under Uncertainty |  | Develops a formal rulebook framework that evaluates system trajectories under stochastic environment responses, incorporating risk measures and hierarchical priorities among objectives. The resulting preorder over trajectories supports explainable selection of navigation and interaction behaviors in uncertain environments such as autonomous driving. | https://arxiv.org/abs/2603.04603 |
| GAIDE: Graph-based Attention Masking for Spatial- and Embodiment-aware Motion Planning |  | Introduces a neural informed sampler that encodes robot embodiment and scene geometry as a graph, integrating it into a transformer via attention masking to bias sampling-based planners. Demonstrations show higher planning success and efficiency in high-dimensional manipulator motion planning compared to uniform, hand-crafted, and prior neural samplers. | https://arxiv.org/abs/2603.04463 |
| Many-RRT*: Robust Joint-Space Trajectory Planning for Serial Manipulators |  | Extends RRT*-Connect to plan concurrently toward multiple joint-space IK solutions of a single end-effector goal, treating each as a separate goal tree. | Direct Contact-Tolerant Motion Planning With Vision Language Models |  | Proposes a direct contact-tolerant navigation framework where a VLM-based point cloud partitioner labels contact-allowable regions from images and propagates these semantics to 3D scans. | VPWEM: Non-Markovian Visuomotor Policy with Working and Episodic Memory |  | Introduces a diffusion-based visuomotor policy with dual memory: short-term working memory over recent observations and a transformer compressor that distills older history into compact episodic tokens. This architecture supports nearly constant per-step compute while outperforming VLAs and standard diffusion policies on long-horizon mobile manipulation and manipulation benchmarks. | https://arxiv.org/abs/2603.04910 |
