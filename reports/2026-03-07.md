## Executive Summary

- 16 of 66 papers match the user’s institution/topic filters, with strong coverage of motion planning, mobile manipulation, dexterous/whole‑body control, and core tools (OMPL, Koopman-based MPC, informed sampling).
- Motion planning and control see notable advances: GPU-accelerated whole‑body planning for high‑DoF robots, multi-goal RRT* for manipulators, distributionally robust collision avoidance, Koopman-accelerated sampling-based control, and graph-informed sampling for complex workspaces.
- Mobile and mobile manipulation work emphasizes risk awareness and long-horizon behavior: risk-sensitive RL for mobile manipulators, semantic frontier-based navigation, and waypoint prediction with calibrated uncertainty, plus mine-ready quadruped navigation.
- Dexterous / complex manipulation is pushed by sim-to-real tactile latent distillation, large-scale bimanual dexterous grasping datasets, and memory-augmented visuomotor policies for long-horizon, non-Markovian tasks.
- Vision-language(-action) models are probed and extended: interpretability and linear feature control for VLAs, benchmarking memory in VLA policies, hierarchical diffusion policies refined on-policy, and adaptive complexity-aware routing for VLA inference.

| Loop Closure via Maximal Cliques in 3D LiDAR-Based SLAM | Massachusetts Institute of Technology; University of Bonn; Universidad Politécnica de Madrid; others | Proposes CliReg, a deterministic loop-closure validator that replaces RANSAC with a maximal-clique search over a compatibility graph of 3D feature correspondences. By seeking the largest mutually consistent subset of matches, it avoids random sampling failures and yields more robust pose estimation in noisy, ambiguous, or sparse LiDAR data. Integrated into a real-time LiDAR SLAM pipeline using binary 3D descriptors and Hamming-distance tree matching, it consistently reduces pose error and improves loop reliability across multiple datasets. Additional tests on 2D projection maps show the approach generalizes to different spatial representations. | https://arxiv.org/abs/2603.05397 |
| Accelerating Sampling-Based Control via Learned Linear Koopman Dynamics | ETH Zurich; University of Massachusetts Amherst; others | Introduces MPPI-DK, which replaces nonlinear dynamics in Model Predictive Path Integral control with a deep Koopman operator that provides a learned linear approximation. The Koopman model is trained from interaction data, enabling fast rollout and dense sampling without analytic dynamics. Simulation on pendulum and surface vessel, and hardware tests on a quadruped, show control performance close to true-dynamics MPPI with substantially reduced computation, enabling real-time deployment. This offers a practical path to sample-based optimal control for systems with complex or unknown dynamics. | https://arxiv.org/abs/2603.05385 |
| OpenFrontier: General Navigation with Visual-Language Grounded Frontiers | ETH Zurich; Microsoft | Frames navigation as selecting and reaching sparse frontier subgoals that are semantically anchored using vision-language priors. Rather than dense 3D mapping or trained policies, OpenFrontier uses off-the-shelf VLM/VLA models to score frontiers with respect to language goals and selects promising ones for exploration. The system is training-free and avoids model fine-tuning, yet achieves strong zero-shot performance on multiple navigation benchmarks. Real-world mobile robot deployments confirm that semantically grounded frontiers enable efficient, goal-directed exploration. | https://arxiv.org/abs/2603.05377 |

A whole-body teleoperation system generates data on full-body coordination, allowing learning of an end-to-end policy. Experiments in sim and on real humanoids show robust manipulation across wide areas and clutter, outperforming egocentric depth-based baselines that suffer from FOV and occlusion issues. | https://arxiv.org/abs/2603.05355 |

LPS thus balances return maximization with dataset support without heavy hyperparameter tuning. Across OGBench and real-world robotic datasets, it achieves state-of-the-art performance, surpassing behavioral cloning and prior latent-steering baselines. | https://arxiv.org/abs/2603.05296 |

Benchmarks on S² show improved trajectory accuracy and lower path deviation versus existing manifold DS methods. Demonstrations on a manipulator and a mobile manipulator show online adaptation of poses and damping matrices on SE(3)/SPD(n) for safe, geometry-aware motion. | https://arxiv.org/abs/2603.05268 |
