| Trajectory Planning for Safe Dual Control with Active Exploration | University of Michigan | Introduces Dual-gatekeeper, a framework that couples robust trajectory planning with active exploration under explicit safety and task-cost budgets. Exploration is only executed when it can be certified to improve model uncertainty without violating safety or mission-level performance, and is validated on quadrotor navigation and autonomous car racing under parametric uncertainty. | https://arxiv.org/abs/2604.15507 |
| One-Shot Cross-Geometry Skill Transfer through Part Decomposition | Brown University | Proposes a compositional approach to skill transfer that decomposes objects into semantic parts and uses generative shape models to align interaction points between demonstration and novel objects. This enables one-shot transfer of manipulation skills to objects with significantly different geometries, demonstrated in both simulation and real-world tasks. | https://arxiv.org/abs/2604.15455 |
| $\pi_{0.7}$: a Steerable Generalist Robotic Foundation Model with Emergent Capabilities |  | Presents $\pi_{0.7}$, a generalist robotic foundation model conditioned on rich multimodal prompts (language, metadata, subgoal images) to steer behavior and strategy. The model demonstrates zero-shot generalization across embodiments and complex household manipulation tasks, achieving performance comparable to specialized RL-finetuned policies on cases like espresso machine operation and laundry folding. | https://arxiv.org/abs/2604.15483 |
| VADF: Vision-Adaptive Diffusion Policy Framework for Efficient Robotic Manipulation |  | Introduces VADF, a model-agnostic framework that accelerates diffusion-policy training and inference for manipulation via a loss-predicting network for hard-negative mining and a hierarchical vision-based task segmenter. The approach adaptively allocates noise and execution schedules to simple vs. complex sub-tasks, improving convergence speed and early success rates while reducing compute. | https://arxiv.org/abs/2604.15938 |
| Contact-Aware Planning and Control of Continuum Robots in Highly Constrained Environments |  | Develops a planning and control pipeline for continuum robots that explicitly evaluates and penalizes hazardous contacts while allowing benign ones in constrained anatomical environments. The method yields kinematically feasible trajectories and contact-aware Jacobians, achieving 100% target reach without dangerous tip contact and sub-2 mm tracking error in hardware tests on patient-specific anatomical models. | https://arxiv.org/abs/2604.15638 |
| ShapeGen: Robotic Data Generation for Category-Level Manipulation |  | Proposes ShapeGen, a simulator-free, 3D data generation pipeline for category-level manipulation that learns functional warpings between shapes to curate a reusable shape library. Using this library, it synthesizes physically plausible, function-aware demonstrations with minimal human input, significantly improving policies’ robustness to intra-category shape variation in real-world evaluations. | https://arxiv.org/abs/2604.15569 |
| Continual Hand-Eye Calibration for Open-world Robotic Manipulation |  | Presents a continual learning framework for hand-eye calibration that resists catastrophic forgetting across sequentially encountered scenes. A spatial-aware replay strategy maintains a geometrically diverse buffer, while dual distillation separately preserves global scene layout and precise pose knowledge, sustaining accuracy on prior scenes while adapting to new ones. | https://arxiv.org/abs/2604.15814 |
| Safe and Energy-Aware Multi-Robot Density Control via PDE-Constrained Optimization for Long-Duration Autonomy | University of Waterloo | Formulates multi-robot density control using the Fokker–Planck PDE to capture stochastic motion, integrating control Lyapunov and barrier functions to enforce target density tracking, obstacle avoidance, and energy sufficiency over charging cycles. The resulting quadratic program can run in real time and is validated in multi-robot experiments and simulations under localization and motion uncertainty. | https://arxiv.org/abs/2604.15524 |
