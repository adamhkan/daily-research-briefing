Executive Summary

  • Multiple works push the frontier of motion planning under constraints, including SIMD-accelerated manifold projection for real-time whole-body planning and hierarchical neural value fields guided by sparse roadmaps.
  • Several papers advance long-horizon manipulation and mobile manipulation via hierarchical VLA architectures, episodic spatial memory, and reflection-based planning agents.
  • Safety-focused methods gain traction, with new sampling strategies for learned safety filters, robust Q-function-based CBFs for black-box systems, and capability-aware multi-robot CBF formulations.
  • Motion planning in structured domains sees improvements through scale-invariant sampling for disassembly and a compositional framework (Mosaic) combining rule-based and learned planners for driving.
  • Institution-linked work from Stanford, UW, and CMU spans learned safety filters, multi-agent responsibility modeling, and uncertainty-aware radar-inertial odometry.

Boundary Sampling to Learn Predictive Safety Filters via Pontryagin's Maximum PrincipleStanford UniversityIntroduces a PMP-based characterization of near-violation boundary trajectories to guide data collection for learned Hamilton-Jacobi reachability. The resulting Control Barrier Value Function yields more sample-efficient, accurate safety filters with millisecond-scale runtime, validated in shared-control automotive racing.Link
Learning Probabilistic Responsibility Allocations for Multi-Agent InteractionsUniversity of WashingtonProposes a CVAE-based model to infer multimodal distributions over responsibility allocation in multi-agent scenarios using a differentiable optimization layer that links latent responsibilities to observed controls. Evaluated on INTERACTION driving data, it provides both strong predictive accuracy and interpretable insights into interactive behaviors.Link
UNRIO: Uncertainty-Aware Velocity Learning for Radar-Inertial OdometryCarnegie Mellon UniversityPresents UNRIO, a transformer-based radar-inertial odometry system that learns ego-velocity directly from raw 4D radar spectra with calibrated uncertainty estimates. Integrated into a sliding-window pose graph with IMU, it significantly outperforms DSP baselines, especially under challenging lateral-motion conditions.Link

Vectorizing Projection in Manifold-Constrained Motion Planning for Real-Time Whole-Body ControlPurdue UniversityDevelops a SIMD-accelerated manifold-constrained motion planner that parallelizes projection-based constraint satisfaction. Achieving 100–1000× speedups over prior methods, it enables real-time, constraint-satisfying whole-body motion planning for complex systems like humanoids.Link
Weakly-supervised Learning for Physics-informed Neural Motion Planning via Sparse RoadmapPurdue UniversityIntroduces Hierarchical Neural Time Fields, combining sparse roadmap-derived global supervision with PDE-based local regularization to learn value functions for motion planning. Demonstrated across 18 Gibson environments and hardware, it improves robustness and amortized planning in complex multi-room settings.Link
Scale-Invariant Sampling in Multi-Arm Bandit Motion Planning for Object ExtractionProposes a grow-shrink, scale-invariant sampling scheme embedded in a multi-arm bandit RRT for disassembly-style object extraction in tight spaces. By adaptively discovering useful sampling scales and PCA-based directions, it boosts success rates by an order of magnitude on most tested 3D scenarios.Link
Capability-Aware Heterogeneous Control Barrier Functions for Decentralized Multi-Robot Safe NavigationPresents CA-HCBF, unifying heterogeneous robot dynamics into a canonical second-order form and introducing a directional capability metric for fair safety responsibility allocation. Simulations with up to 30 robots and physical experiments show improved safety and reduced deadlock in dense, mixed-kinematics teams.Link
Synthesis and Deployment of Maximal Robust Control Barrier Functions through Adversarial Reinforcement LearningJohns Hopkins University; Princeton UniversityShows that the safety value function of the Isaacs equation defines a robust discrete-time CBF over the maximal robust safe set and lifts it to a Q-function-based Q-CBF for black-box systems. Using adversarial RL, the method learns less conservative safe sets and enforces safety on nonlinear systems like a quadruped under adversarial disturbances.Link
Mosaic: An Extensible Framework for Composing Rule-Based and Learned Motion PlannersKarlsruhe Institute of TechnologyIntroduces Mosaic, which arbitrates among heterogeneous motion planners via graphs that decouple trajectory generation, verification, and scoring. On nuPlan and interPlan driving benchmarks, it achieves state-of-the-art closed-loop scores and markedly reduces collisions by leveraging complementary rule-based and learned planners without retraining.Link
Goal2Skill: Long-Horizon Manipulation with Adaptive Planning and ReflectionProposes a dual-system framework where a VLM-based high-level planner maintains structured memory and decomposes goals, while a diffusion-based VLA controller executes subtasks from filtered observations. On RMBench, it significantly outperforms prior methods for partial-observability, long-horizon manipulation, highlighting the value of memory and explicit recovery.Link
HiVLA: A Visual-Grounded-Centric Hierarchical Embodied Manipulation SystemIntroduces HiVLA, separating VLM-based high-level task decomposition and visual grounding from a flow-matching Diffusion Transformer action expert with cascaded cross-attention. The hierarchy preserves zero-shot reasoning while improving low-level execution, yielding strong gains in long-horizon and fine-grained manipulation tasks in clutter.Link
ESCAPE: Episodic Spatial Memory and Adaptive Execution Policy for Long-Horizon Mobile ManipulationPresents ESCAPE, combining a persistent, depth-free 3D spatial memory with memory-driven target grounding and an adaptive execution policy that interleaves global navigation and local manipulation. It attains state-of-the-art ALFRED performance and robust long-horizon mobile manipulation even with reduced instruction guidance.Link
UMI-3D: Extending Universal Manipulation Interface from Vision-Limited to 3D Spatial PerceptionExtends the wrist-mounted Universal Manipulation Interface with lightweight LiDAR and a unified calibration pipeline to enable LiDAR-centric SLAM and robust 3D demonstration capture. The improved data quality enhances downstream visuomotor policies and enables challenging tasks like deformable and articulated object manipulation.Link
