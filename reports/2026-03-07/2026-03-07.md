## Executive Summary

- Dexterous and bimanual manipulation see strong progress via sim-to-real tactile distillation (PTLD) and a 20M-frame synthetic dataset for universal bimanual grasps (UltraDexGrasp).
- Motion planning advances span dynamics-aware GPU stacks for high-DoF robots (cuRoboV2), multi-goal joint-space sampling (Many-RRT*), and graph-structured informed sampling (GAIDE).
- Humanoid and mobile manipulation are tackled with physics-aware whole-body VLA control (PhysiFlow), omnidirectional LiDAR-based policies for large-workspace humanoid manipulation (Omni-Manip), and risk-aware visuomotor RL for mobile manipulation.
- Memory and horizon scaling for manipulation improve through working/episodic memory diffusion policies (VPWEM), self-evolving diffusion policies (SeedPolicy), and a standardized VLA memory benchmark (RoboMME).
- VLAs and LLMs are increasingly controlled and leveraged as tools: interpretable feature steering in VLAs, tri-system VLA frameworks with critics, adaptive Act/Think/Abstain routing, and in-context policy learning via code rewriting (AOR).
- Digital twins, uncertainty-aware navigation, and medical/contact-rich planning extend planning and control methods into real-world constrained domains, including cochlear implantation and textile sorting cells.
