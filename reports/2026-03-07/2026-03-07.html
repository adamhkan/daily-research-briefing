Executive Summary

  • Dexterous and bimanual manipulation see strong progress via sim-to-real tactile distillation (PTLD) and a 20M-frame synthetic dataset for universal bimanual grasps (UltraDexGrasp).
  • Motion planning advances span dynamics-aware GPU stacks for high-DoF robots (cuRoboV2), multi-goal joint-space sampling (Many-RRT*), and graph-structured informed sampling (GAIDE).
  • Humanoid and mobile manipulation are tackled with physics-aware whole-body VLA control (PhysiFlow), omnidirectional LiDAR-based policies for large-workspace humanoid manipulation (Omni-Manip), and risk-aware visuomotor RL for mobile manipulation.
  • Memory and horizon scaling for manipulation improve through working/episodic memory diffusion policies (VPWEM), self-evolving diffusion policies (SeedPolicy), and a standardized VLA memory benchmark (RoboMME).
  • VLAs and LLMs are increasingly controlled and leveraged as tools: interpretable feature steering in VLAs, tri-system VLA frameworks with critics, adaptive Act/Think/Abstain routing, and in-context policy learning via code rewriting (AOR).
  • Digital twins, uncertainty-aware navigation, and medical/contact-rich planning extend planning and control methods into real-world constrained domains, including cochlear implantation and textile sorting cells.

PTLD: Sim-to-real Privileged Tactile Latent Distillation for Dexterous ManipulationCarnegie Mellon University; UC Berkeley; University of WashingtonIntroduces PTLD, a sim-to-real framework that avoids tactile simulation by using privileged real-world sensors to supervise a tactile state estimator, which augments proprioception-only policies. On in-hand rotation and reorientation benchmarks, the distilled tactile representation dramatically boosts success over proprioceptive baselines, enabling more robust dexterous manipulation.Link
Observing and Controlling Features in Vision-Language-Action ModelsStanford UniversityAnalyzes internal VLA representations to identify linearly decodable features and then uses minimal linear interventions, framed as an optimal control problem, to steer behaviors without fine-tuning. Demonstrates that such targeted edits can reliably adjust robot actions in closed loop, enabling real-time alignment of VLA policies to user preferences.Link
RoboMME: Benchmarking and Understanding Memory for Robotic Generalist PoliciesStanford UniversityProposes RoboMME, a 16-task benchmark for long-horizon, history-dependent manipulation focused on different memory types (temporal, spatial, object, procedural). By evaluating 14 memory-augmented variants of a generalist VLA backbone, the work shows that memory design choices are highly task-dependent, clarifying where different representations help or fail.Link

cuRoboV2: Dynamics-Aware Motion Generation with Depth-Fused Distance Fields for High-DoF RobotsNVIDIAPresents a unified GPU-native motion generation stack combining B-spline trajectory optimization with torque limits, fast TSDF/ESDF distance fields, and scalable whole-body kinematics/dynamics. The system delivers large speedups and near-perfect collision-free success on arms and humanoids, including 48-DoF setups where prior GPU planners fail.Link
PhysiFlow: Physics-Aware Humanoid Whole-Body VLA via Multi-Brain Latent Flow Matching and Robust TrackingShanghai Jiao Tong UniversityIntroduces a physics-aware multi-brain VLA framework that fuses semantic intent with whole-body control for humanoids. Through latent flow matching and robust tracking, the system achieves stable, vision-language-guided full-body coordination on dynamic tasks.Link
Omni-Manip: Beyond-FOV Large-Workspace Humanoid Manipulation with Omnidirectional 3D PerceptionProposes Omni-Manip, a LiDAR-driven 360° visuomotor policy that uses time-aware attention pooling over panoramic point clouds for large-workspace humanoid manipulation. A whole-body teleoperation system provides training data, and experiments show strong robustness in cluttered, beyond-FOV scenarios versus RGB-D baselines.Link
UltraDexGrasp: Learning Universal Dexterous Grasping for Bimanual Robots with Synthetic DataChinese University of Hong Kong; e Chinese University of Hong KongBuilds UltraDexGrasp, a pipeline that synthesizes 20M-frame multi-strategy bimanual grasp trajectories via optimization and planning to train a point-cloud-based grasp policy. Trained purely in simulation, the policy zero-shot transfers to real bimanual robots, achieving over 80% success on diverse novel objects.Link

This adaptive routing reduces expensive VLM calls while improving robustness and autonomy on challenging long-horizon manipulation benchmarks.Link

This nearly constant-cost memory allows better handling of long-term dependencies, yielding >20% gains on memory-intensive manipulation tasks and improvements on mobile manipulation.Link
